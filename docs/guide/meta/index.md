# @svaio/meta

Zero-config Meta plugin for Sveltekit (seo, meta tags, opengraph, jsondl...)

::: code-group

```bash [npm]
npm install @svaio/meta
```

```bash [pnpm]
pnpm install @svaio/meta
```

```bash [yarn]
yarn add @svaio/meta
```

```bash [bun]
bun add @svaio/meta
```

```bash [deno]
deno add @svaio/meta
```

:::
